# Student study Group Finder 
**Uganda Christian UNiversity | Easter Semester Examination 2026

##Introduction
The **Student Group Finder** is a web-based application designed to enhance collaborative learning at UCU. It allows Students to discover, create, and organise study groups for specific courses, breaking down the barriers of limited personal networks and closed Whatapp groups.
**
## Key Features
* **User Management:** Secure Student registration and login.
* **Group Creation:** Students can form groups by course name, code, and location.
* **Repository & Search:** Browse and filter available study groups by faculty or course.
* **Study Sessions:** Schedule meetings with specific dates and times.
* **Dashboard:** Personal overview of joined groups and upcoming sessions. 

## Tech Stack
* **Frontend:** React.js
* **Backend:** Node.js (Express)
* **Database** MYSQL
* **Authentication:** JWT (JSON Web Tokens)
